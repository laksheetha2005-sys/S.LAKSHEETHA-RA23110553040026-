This project focuses on performing customer segmentation analysis on a telecom dataset using data analytics techniques. The main objective is to divide customers into different groups based on attributes such as gender, senior citizen status, contract type, payment method, and other service-related features.

The dataset is preprocessed to handle missing values, remove duplicates, and ensure data consistency. Using Python libraries like Pandas, customers are grouped into meaningful categories to identify patterns and trends in the data. Visualization techniques such as bar charts and pie charts are used to represent customer distribution clearly.

This analysis helps in understanding customer behavior and provides insights that can support better decision-making, targeted marketing, and improved customer retention strategies.
